Minha Experiência com Ferramentas de Controle de Versão

No início, o uso do Git pareceu um pouco complexo devido à necessidade de utilizar comandos via terminal, como entender a diferença prática entre o git add e o git commit. Outra dificuldade inicial foi realizar a configuração das chaves SSH para conectar o computador ao GitHub com segurança.

Esses problemas foram resolvidos através da leitura da documentação, acompanhamento de tutoriais passo a passo e, principalmente, pela prática constante dos comandos no Git Bash. Com o tempo, o fluxo de trabalho ficou mais natural, tornando evidente como o Git facilita a organização e o histórico do código.
